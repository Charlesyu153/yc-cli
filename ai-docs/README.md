# AI 文档管理系统

YC-CLI 的 AI 文档管理系统，用于任务管理和上下文连续性。

## 核心功能

- 任务文档管理（创建、更新、归档）
- 上下文恢复（长期任务连续性）
- AI 协作规范（统一的工作流程）
- 工具脚本（自动化任务管理）

## 快速使用

```bash
# 创建新任务
cd templates
./new-task.sh "任务名称" "负责人"

# 归档已完成任务
./archive-task.sh "任务名称"

# 导出上下文
./export-context.sh "任务名称"
```

## 核心规范

1. 禁止使用表情符号
2. 每完成一步就 commit
3. 任务完成后归档
4. 代码文件不超过 500 行
